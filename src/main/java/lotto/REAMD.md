# 2단계 - 로또(자동)
- 로또 구입 금액을 입력하면 구입 금액에 해당하는 로또를 발급해야 한다.
- 로또 1장의 가격은 1000원이다.
- 구매한 로또의 현황을 출력한다.
- 지난주 당첨 번호를 입력하면 당첨 통계와 수익률을 표시한다.

# 3단계 - 로또(2등)
- 2등을 위해 추가 번호를 하나 더 추첨한다.
- 당첨 통계에 2등도 추가해야 한다.
